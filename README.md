# Pokemon Data Analysis Project

This project analyzes a Pokemon dataset using Python and performs basic data analysis and visualization.

## Tools Used
- Python
- Pandas
- Matplotlib

## Project Features
- Data loading from CSV file
- Data exploration
- Filtering Pokemon by attack
- Average attack by Pokemon type
- Graph visualization
- Strongest and fastest Pokemon analysis

## Dataset
Pokemon dataset containing attributes such as Attack, Defense, Speed, Type, and Legendary status.
